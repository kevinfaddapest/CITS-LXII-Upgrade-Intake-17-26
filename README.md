
# CITS-LXII Upgrade Intake 17–26

## 📌 Project Overview

This project is a web-based system developed as part of the **CITS LXII Upgrade Intake (17–26)** program. It is designed to manage and streamline system operations efficiently.

---

## 🚀 Features

* User authentication and authorization
* Database management with SQLite
* Static file handling
* Logging system
* Modular application structure

---

## 🛠️ Technologies Used

* Python
* Django Framework
* SQLite Database
* HTML, CSS, JavaScript

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/kevinfaddapest/CITS-LXII-Upgrade-Intake-17-26.git
```

2. Navigate into the project directory:

```
cd CITS-LXII-Upgrade-Intake-17-26
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run migrations:

```
python manage.py migrate
```

5. Start the development server:

```
python manage.py runserver
```

---

## ▶️ Usage

* Open your browser and go to:

```
http://127.0.0.1:8000/
```

* Use the system interface to interact with the application.
